# Bewerbungsmanagement — Hub-App
**Stand:** 14. Mai 2026 | **Version:** Beta ✅

## App-URLs
- **GitHub:** https://github.com/waltermarcd-dot/bewerbungsmanagement

## Beschreibung
Zentraler Bewerbungs-Hub — Nutzer wählen auf der Startseite ob sie ihren **Lebenslauf verbessern** oder ein **Anschreiben erstellen** möchten und werden direkt in den jeweiligen Flow geleitet.

## Routen
| Route | Beschreibung |
|---|---|
| `/` | Hub-Startseite — Auswahl Lebenslauf / Anschreiben |
| `/lebenslauf` | Lebenslauf-Optimierer Flow |
| `/anschreiben` | Anschreiben-Generator Flow |
| `/admin/vouchers` | Admin-Dashboard |

## Features
- Einheitliches Design (M Corp Apps)
- Login nur beim Download — maximale Benutzerfreundlichkeit
- Google + Email/Password Login
- Voucher-System für B2B-Partner
- Stripe-Zahlung (10 €)
- DSGVO-konform, lokale Fonts
- EU AI Act Transparenzhinweis
